My Work

Recent Work

Game-Go

GameGo is a full-stack web app to help users find and save new video games they want to play. This isn't like any old video game search, however. With GameGo, you input your game criteria and we'll server you up random games that match. Find a game you like and saved it to your own list to come back and view at any time. The app is built using express.js, jQuery, mySQL, sequelize, handlebars.js, Bootstrap, and the RAWG API.

Eat-Da-Burger

Eat-Da-Burger is a restaurant app that lets users input the names of burgers they'd like to eat. The devour button also allows users to keep track of burgers they've already tried! The app uses express.js, mySQL, a custom ORM, jQuery, handlebars.js, and Materialze.

Note Taker App

The Note Taker App allows users to create and save notes all in one convenient place. This simple app is built on express.js and follows the CRUD method, allowing users to create, update, and delete their notes.

What's For Dinner?

Can't figure out whether you want to cook with the things you have at home or go out to eat? The What's For Dinner app can help you choose. Search for recipes by ingredient based on what you have at home. If you don't find anything you like search for restaurants by cuisine wherever you are! The app is built using HTML5, CSS3, Materialize, JavaScript, jQuery, and two third party APIs: Spoonacular and Zomato.
